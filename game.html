Memory game

::

Rules

  • You will start the game by flipping over one card
  • If the next card you flip matches with the previous one then a pop up alert notifies you and you get +1 to your score
  • These 2 cards will then disappear.
  • If the next card you flip does not match, a pop up alert notifies you of this and the cards flip back
  • The game continues until you match all the cards on the board
